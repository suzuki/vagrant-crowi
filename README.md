# vagrant-crowi

create development server for [Crowi](https://github.com/crowi/crowi).

## Requirements

- [Vagrant](https://www.vagrantup.com/)
- [Ansible](http://www.ansible.com/)

## Usage

```
$ vagrant up
```

Sometimes it will fail. So,

```
$ vagrant provision
```

## Where is Crowi?

- on VM
  - `/vagrant/crowi/`

- on Host
  - `./crowi/`

These are Vagrant's 'synced_folder'.
